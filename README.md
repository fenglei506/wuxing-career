# 五行八卦 · 职业命理分析网页

## 项目结构

```
fortune-career-web/
├── index.html      # 主页面
├── style.css       # 样式文件（简约移动端风格）
├── bazi.js         # 八字排盘核心算法
├── career.js       # 职业分析逻辑
├── app.js          # 主应用交互逻辑
├── placeholder-qr.png  # 占位收款码（需要替换）
└── README.md       # 说明文档
```

## 功能特性

### 1. 八字排盘
- 年柱计算（立春为界）
- 月柱计算（节气为界）
- 日柱计算（基准日推算法）
- 时柱计算（五鼠遁日起时诀）

### 2. 五行分析
- 五行分布统计（金/水/木/火/土）
- 身强弱判断（得月令、地支根气、天干帮扶）
- 用神取法（扶抑法）

### 3. 职业匹配
- 日主性格特质分析
- 五行行业匹配（最旺五行 → 推荐行业）
- 身强弱调整建议
- 方位、颜色等开运建议

### 4. 支付功能
- 当前为占位收款码
- 后期可替换为第三方支付平台集成

## 技术说明

### 八字算法来源
- 基于 `bazi-fortune` skill 的 calculation-rules.md
- 严格遵循子平真诠体系

### 职业分析来源
- 基于 `fortune-career` skill 的 career_analysis.py
- 五行行业映射、十神职业倾向

## 使用方式

### 本地测试
```bash
cd fortune-career-web
# 直接在浏览器打开 index.html
# 或使用本地服务器
python3 -m http.server 8080
```

### 部署到线上
1. 上传所有文件到服务器
2. 替换 placeholder-qr.png 为真实收款码
3. 如需自动化支付，集成第三方支付API

## 后续优化建议

1. **精确节气计算**：当前使用近似日期，可接入精确节气时刻数据
2. **紫微斗数**：可增加命宫主星分析（当前仅用八字）
3. **第三方支付**：集成易支付、虎皮椒、Stripe等支付平台
4. **详细报告**：付费解锁更详细的大运流年分析

## 收款码替换说明

将 `placeholder-qr.png` 替换为你的收款码：
- 支付宝个人收款码
- 微信个人收款码
- 第三方支付平台二维码
