## 核心算法

### 八字排盘 (bazi.js)
- 年柱：立春为界 (2月4日)
- 月柱：节气分界 (简化近似日期)
- 日柱：基准日推算法 (1900年1月1日 = 甲戌日)
- 时柱：五鼠遁日起时诀

### 职业分析 (career.js)
- 日主性格特质 (10天干 → 性格类型)
- 五行行业匹配 (最旺五行 → 推荐行业)
- 身强弱调整 (得月令、地支根气、天干帮扶)

## 待解决/优化

### 🔴 支付验证 (优先级高)
当前：用户点击"已完成支付"直接解锁报告，无验证机制

**天津银行智慧商户通 API 需获取的信息**：
1. 商户号 (Merchant ID)
2. AppID / 应用ID
3. AppKey / 密钥 (签名验证)
4. 商户私钥 (证书签名)
5. API 接口地址 (下单、查询、回调)
6. 商户平台后台账号 (确认 API 权限已开通)

**实现流程**：
- 用户点击支付 → 后端调用下单API → 返回支付链接
- 用户扫码支付 → 银行回调服务器
- 服务器验证支付成功 → 前端解锁报告
